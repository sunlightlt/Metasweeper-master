# 开发计划
计划开发：

- 能计算开空概率的局面分析工具。
- 多种语言的切换。
- 录像文件的签名认证。
- 在本地记录个人纪录。
- 防止在本软件作弊以取得异常成绩。
- 自动更新。
- 可能开发双人扫雷。

而不会开发：
- 移动端的扫雷。
- 在本地统计个人数据的功能。
- 安装程序和启动界面。
- 扫雷游戏以外的游戏、无猜模式以外的扫雷模式。
- 非标准且花哨的ui、任何多余且匪夷所思的选项。
- 任何用于在其他软件作弊以取得异常短的时间的成绩的设计。
- 扫码收费以解锁某功能的功能。
